To set up a ctable server,

create your ctables

    package require ctable_server

::ctable_server::register each ctable

::ctable_server::register_instantiator each ctable creator, if any, that you want clients to invoke to create new instances of 
ctables wtih.

vwait or do something to be in the Tcl event loop.


    #
    # demo server
    #
    #

    source demo2.ct
    package require Cable

    package require ctable_server

    ::ctable_server::register_instantiator cable_info

    proc doit {} {
	::ctable_server::setup

    vwait die
    }

    if !$tcl_interactive doit
